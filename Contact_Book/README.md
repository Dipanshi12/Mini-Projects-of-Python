#  Contact Book Management System

A simple Contact Book Management System built using Python. This project is menu-driven and uses functions and dictionaries to manage contacts.

##  Features

-  Add a new contact
-  Display all contacts
-  Search for a contact by name
-  Delete a contact
-  Exit the application

##  Technologies Used

- Python 3
- Functions
- Dictionary
- Loops
- Conditional Statements

##  Project Structure

```
contact_book.py
README.md
```

##  How to Run

1. Clone the repository:
   ```bash
   git clone <repository-link>
   ```

2. Navigate to the project folder:
   ```bash
   cd contact-book
   ```

3. Run the program:
   ```bash
   python contact_book.py
   ```
