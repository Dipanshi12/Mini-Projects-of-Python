# Employee Management System

## Project Description
The Employee Management System is a simple Python console-based application that allows users to manage employee records. It provides features to add, search, update, delete, and display employee information using Python dictionaries and functions.

## Features
- Add a new employee
- Search an employee by name
- Update employee salary
- Delete an employee
- Display all employee records
- Menu-driven interface

## Technologies Used
- Python 3
- Dictionary
- Functions
- Loops
- Conditional Statements

## Project Structure
Employee_Management_System.py

## How to Run
1. Install Python 3.
2. Download or clone this repository.
3. Open the project in VS Code or any Python IDE.
4. Run the Python file.

```bash
python Employee_Management_System.py
```
